## How to search

1、使用 [Sourcegraph.com](https://sourcegraph.com/github.com/ruanyf/weekly) 进行搜索。（推荐方法）

2、使用 GitHub 自带的搜索，但只支持搜索英文单词。

3、将这个仓库克隆到本地，然后在仓库目录使用下面的命令。

`$ grep -nri [搜索词] essays | cat --number`
